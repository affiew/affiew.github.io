Leetcode 142. Linked List Cycle II

https://leetcode.com/problems/linked-list-cycle-ii/

没想出来 只知道快慢指针找环不知道怎么找环的入口

随想录：

  • fast指针一定先进入环中，如果有环fast指针和slow指针一定是在环中相遇
  • 假设从头结点到环形入口节点为x，环形入口节点到相遇节点为y, 从相遇节点到环形入口节点为z

  • 相遇时： slow指针走过 x + y，fast指针走过 x + y + n (y + z)，n为fast在环内走了n圈， y+z为 一圈内节点的个数。

  • fast指针走过的节点数 = slow指针走过的节点数 * 2：(x + y) * 2 = x + y + n (y + z)

  • x = n (y + z) - y = (n - 1) (y + z) + z

  • n == 1 时 x = z

  • 头结点出发一个指针，相遇节点也出发一个指针，这两个指针每次只走一个节点，当这两个指针相遇的时就是环形入口节点。

好难

今日收获

学习时长2小时

虚拟头节点 快慢双指针 链表对齐 环形链表

不写注释不是好习惯 明天改

Leetcode 704. Binary Search

https://leetcode.com/problems/binary-search/

  1. 之前的写法（左闭右闭）
  • [left, right] 要用 <=
  • right = middle - 1
  1. 左闭右开
  • [left, right) 要用 <
  • right = middle

总结： 区间的定义是不变量，在循环中坚持根据查找区间的定义来做边界处理，就是循环不变量规则。
